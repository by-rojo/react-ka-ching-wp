== Description ==

Best WordPress plugin to generate and customize HTML & XML sitemaps for your website.

HTML sitemaps are helpful to website visitors to navigate and find pages on your website, while XML sitemaps help search engines to crawl your website which leads to higher SEO rankings.

Main plugin features include:

- HTML Sitemap,
- XML Sitemaps including Video Sitemaps,
- Custom Post Types support,
- WooCommerce support,
- Blacklist and custom sorting,
- Adding custom links,
- Generate and Edit robots.txt file,
- and much more...

== Installation ==

= AUTOMATIC INSTALLATION =

Automatic installation method is the easiest and most recommended option.

1. Log in to your WordPress Dashboard, navigate to the Plugins menu, and click on “Add New”.
2. Enter ‘click5’ (without quotes) in the search field and click the ‘Search Plugins’ button.
3. On the list of search results, click the 'Install Now' button for the 'Sitemap by click5' plugin.
4. Then click the red 'Active' button to finalize the installation.
5. You will be automatically redirected to the plugin Settings page, where you can copy the shortcode and configure both sitemaps - HTML & XML.

= MANUAL INSTALLATION =

1. Download the plugin and unzip the contents. You should see a single folder named 'sitemap-by-click5'.
2. Connect to the FTP server with your WordPress installation.
3. Upload the 'sitemap-by-click5' plugin folder to the /wp-content/plugins/ folder on your FTP server.
4. Continue to step 3 of Automatic Installation instructions above.
